### OpenRouter

OpenRouterは、複数のAIモデルプロバイダーのAPIを統合し、単一のエンドポイントから利用できるサービスです。様々なモデルを比較検討したり、コスト効率の良いモデルを柔軟に選択したりできる点が大きな魅力です。

*   **概要**: OpenAI、Anthropic、Perplexity AIなど、多数のAIモデルプロバイダーを統合したAPIサービスです。ユーザーはOpenRouterのAPIキー一つで、様々なモデルにアクセスできます。
*   **無料枠の目安**: 新規登録時に一定量の無料クレジットが付与されることが多く、モデルやリクエスト数に応じて消費されます。例えば、私が試用した際には、初期で$5相当のクレジットが提供され、小型モデルであれば数千回のリクエストが可能でした（モデルとトークン消費量によって変動）。このクレジットは有効期限が設定されている場合があるので注意が必要です。
*   **主要対応モデル**: Llama 3 (8B/70B), Mixtral 8x7B, Claude 3 Opus/Sonnet/Haiku (一部モデルは有料クレジット消費), Gemma, Yiなど、オープンソースから商用モデルまで幅広く対応しています。利用可能なモデルは頻繁に更新されます。
*   **APIキー取得手順**:
    1.  OpenRouterの公式サイト（`https://openrouter.ai/`）にアクセスし、メールアドレスまたはGitHubアカウントでサインアップします。所要時間は約3分です。
    2.  ログイン後、ダッシュボードの「Keys」セクションに移動し、「Create New Key」をクリックします。
    3.  生成されたAPIキーをコピーして安全に保管します。
*   **利用例**: Pythonの`requests`ライブラリを使って、Llama 3 8B Instructモデルにリクエストを送る例です。

    ```python
    import requests

    OPENROUTER_API_KEY = "YOUR_OPENROUTER_API_KEY" # 取得したAPIキーに置き換える

    headers = {
        "Authorization": f"Bearer {OPENROUTER_API_KEY}",
        "Content-Type": "application/json",
    }

    data = {
        "model": "meta-llama/llama-3-8b-instruct", # 利用したいモデルを指定
        "messages": [
            {"role": "user", "content": "無料で使えるAI APIについて教えてください。"}
        ]
    }

    response = requests.post("https://openrouter.ai/api/v1/chat/completions", headers=headers, json=data)
    print(response.json()['choices'][0]['message']['content'])
    ```

*   **利用例**: NIMは通常、Triton Inference Serverを介して利用されますが、直接APIエンドポイントが提供される場合もあります。Pythonの`requests`を使った一般的なLLMの呼び出し例です。
