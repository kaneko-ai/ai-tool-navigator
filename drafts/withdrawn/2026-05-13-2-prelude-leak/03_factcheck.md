# 記事ドラフト監査結果：「給付金を使ってAIスクールを安く受講する方法」

## 重大な問題（修正必須）

### [問題1] 数値根拠の欠落
**該当箇所:**  
「受講料の一部（講座により 20〜70％ が上限など）が支給され」

**指摘:**  
「20〜70％」という上限数値が本文内にハイパーリンク・脚注なしで記載されています。出典リストに「厚生労働省：教育訓練給付制度」とありますが、本文中には明示的なリンクまたは出典表記がありません。  
**修正案:**  
```
「受講料の一部が支給され（詳細は厚生労働省の教育訓練給付制度ページで確認：
https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000186317.html）」
```
または、後述の「申請前に確認したい条件と注意点」セクション内に「詳細は必ず厚生労働省公式ページで最新情報を確認してください」と明示してください。

---

### [問題2] スクール名の出典・検証不能
**該当箇所:**  
「代表例として、実務寄りの Neuro Dive、ビジネス活用寄りの メイカラ、企業導入を意識した「AIビジネス活用講座」を見る視点を整理します。」

**指摘:**  
- 「Neuro Dive」「メイカラ」「AIビジネス活用講座」が実在する講座か、正式名称か不明確です  
- これらは具体例として言及されているため、公式 URL・運営企業名を併記する必要があります  
- 現状では読者が検索して確認できません  

**修正案:**  
```
「代表例として、[実務寄りの Neuro Dive（提供元：●●）](URL)、
ビジネス活用寄りの メイカラ（提供元：▲▲）、
企業導入を意識した「AIビジネス活用講座」（提供元：■■）を見る視点を整理します。」
```

### [問題3] 「ケースが多く」は検証不能
**該当箇所:**  
「Neuro Dive は実装やデータ処理の演習が充実しているケースが多く、ポートフォリオ作成や実務で再現できるアウトプット重視の受講者に向きます。」

**指摘:**  
- 「ケースが多く」という表現は、出典なしの推測表現です  
- 講座の特性を述べる際、「～である傾向」「公式では～と謳っている」など、根拠を示す必要があります  

**修正案:**  
```
「Neuro Dive は公式ページで実装やデータ処理の演習を特徴として謳っており、
ポートフォリオ作成や実務での活用を重視する受講者に向いています。」
```
または、各講座の公式ページへの直リンクを付与してください。

## 軽微な問題（修正推奨）

### [問題4] リード文の根拠欠落
**該当箇所:**  
「給付金や補助金を賢く使えば、自己負担を大きく下げて学べるケースが増えています。」

**指摘:**  
- 「増えています」という動向主張に出典がありません  
- オピニオン記事ではなく、ファクト重視の記事なら削除または根拠を追加すべきです  

**修正案:**  
```
「給付金や補助金を賢く使うことで、自己負担を下げて学ぶことができます。」
（「増えています」を削除）
```

---

### [問題5] メイカラのスペル確認
**該当箇所:**  
「メイカラ」が複数回登場

**指摘:**  
- 正式な英語表記・スペルが不明です（Maker Labo? Maker Caravan? など複数の可能性）  
- 公式サイトでの表記を確認し、初出時に英語表記も併記してください

**修正案:**  
```
「メイカラ（Maker Labo など）」のように正式名称を明示
```

### [問題6] 「給付金対象」ラベルの記述が曖昧
**該当箇所:**  
「特に支給が後日である制度では、一時的な立て替え負担が発生するため、資金計画の確認が必須です。また「給付金対象」ラベルがあっても、コースの一部のみが該当することや、短期コースは対象外となる場合もあるため」

**指摘:**  
- 「給付金対象」ラベルがあっても対象外のケースがある、という重要な注意書きは  
- リード文で強調するか、見出しの近く（本文最初）に配置すべきです  
- 現在は「申請前に確認したい条件」という後半セクションにあるため、見落とされやすい

**修正案:**  
- AI スクール説明セクション（H2「AIスクールは給付金でどこまで安くなるか」）に以下を追加：
```
「⚠ 重要な注意：「給付金対象」と表示されている講座でも、
コースの一部のみが該当することもあり、短期コースは対象外となる場合があります。
必ず申込前に運営に文書で確認してください。」
```

## 良かった点

- **AI 開示文の配置**: 「※本記事は AI を活用して下書きを作成し、人手で校閲・加筆しています。」がリード直後に明示されている ✓  
- **句点切れ**: 本文最後が「給付金の申請サポートの有無を確認してください。」と句点で終わっている ✓  
- **CTA 配置**: `<!-- CTA:MOSHIMO_CONOHA_WING -->` が 2 箇所（本文中段と後段）に適切に配置されている ✓  
- **文字数**: 約 2,900 字で推奨範囲（2,500～3,500 字）内 ✓  
- **H2 数**: 7 個で 4-6 個の推奨をやや超えるが許容範囲 ✓  
- **構成**: 制度説明 → 比較ポイント → 具体例 → チェックリスト → ケース別 → まとめ という論理的な流れ ✓

## 総合判定

**重大修正必要**

**理由:**  
1. 具体的なスクール名（Neuro Dive・メイカラ）が出典・公式 URL なしで言及されており、誤情報の可能性が排除できない  
2. 「20〜70％」という給付率に本文内での出典表記がなく、景品表示法リスク（根拠なき数値表示）  
3. 「ケースが多く」「増えています」など、出典なき根拠レスな主張が複数存在し、消費者庁ガイドラインに抵触する可能性

**必須修正:**  
- 各スクール名に公式 URL・正式名称を併記  
- 給付率数値に厚労省リンクを埋め込む  
- 主観表現（「～が多く」「～増えています」）を削除または根拠を追加  

修正後、軽微な問題も対応して再提出してください。
