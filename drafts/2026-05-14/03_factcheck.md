## ファクトチェック実施に必要な情報

ファクトチェック・コンプライアンス審査を進めるには、以下が必要です：

| 項目 | 理由 |
|-----|-----|
| **実際の記事ドラフト本文** | HTML / Markdown の本体部分 |
| **トピックインデックス** | 記事の主題特定（例：`7`） |
| **直近30日間の公開済み記事タイトル** | 重複・類似記事の検出 |

## 次のステップ

以下から提供してください：

1. **`drafts/` フォルダの該当ファイル** を教えてください
   ```
   例: drafts/topic_7_draft.md
   ```

2. または、**記事ドラフトそのものをテキスト形式で貼り付け** てください

3. **TOPICS_legacy.py** や **トピック定義ファイル** があれば確認します

提供いただければ、即座に10項目の厳密な審査を実施します。
